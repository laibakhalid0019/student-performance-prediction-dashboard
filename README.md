# Student Performance Prediction Dashboard

A machine learning project that predicts student academic performance (Midterm-I, Midterm-II, and Final Exam marks) using historical assessment data.

## 📌 Overview

This project uses regression models to estimate student scores based on assignments, quizzes, and previous assessments. It also includes an interactive dashboard for exploring predictions and model performance.

## 🚀 Features

* Predict Midterm-I marks
* Predict Midterm-II marks
* Predict Final Exam marks
* Exploratory Data Analysis (EDA) with visualizations
* Data preprocessing & feature engineering
* Multiple regression model comparison
* K-Fold Cross Validation
* Bootstrapping with confidence intervals
* Baseline model comparison
* Interactive Streamlit dashboard

## 🛠 Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn
* Streamlit

## 📊 Models Used

* Linear Regression
* Polynomial Regression
* Multiple Regression
* Dummy Regressor (Baseline)

## 📈 Evaluation Metrics

* MAE (Mean Absolute Error)
* RMSE (Root Mean Squared Error)
* R² Score
